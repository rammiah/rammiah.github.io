Leetcode 318 Maximum Product of Word Lengths

January 1, 2019

题目分析

Given a string array words, find the maximum value of length(word[i]) * length(word[j]) where the two words do not share common letters. You may assume that each word will contain only lower case letters. If no such two words exist, return 0.

Example 1:

Given ["abcw", "baz", "foo", "bar", "xtfn", "abcdef"] Return 16 The two words can be "abcw", "xtfn".

Example 2:

Given ["a", "ab", "abc", "d", "cd", "bcd", "abcd"] Return 4 The two words can be "ab", "cd".

Example 3:

Given ["a", "aa", "aaa", "aaaa"] Return 0 No such pair of words.

题目很简单，就是给你一个字符串的vector，让你判断里面的两个不含有相同字母的字符串的最大长度乘积，说的很清楚了。

思考

知道了题目的意思之后开始思考如何解题。下面是我的思路：

  1. 哈希表记录字符串的字母信息：这个方式的思路就是将所有的字母转化为不同的质数，先判断这个质数是否乘过，把字符串里面的字母映射的数字全部乘到一起。这个方法如果在Python或者Java等整数限制大小或者有对应的类的话还是比较方便的，毕竟在判断是只需要对所有的字母进行取模测试看两个字符串对应的hash是否结果都是0就行了，但是这个方法在c/c++中是会移除的，unsigned long long也没能存储 下前26个质数的乘积，所有我放弃了。

  2. 使用vector<bool>使用大小vector<bool>记录一个字符串的某个字母是否出现，但是这个方式的话时间上并没有太大改进，虽然我使用这个方法过去了，但是只在所有提交中排在后20%，继续改进。

  3. 位操作：上面的vector<bool>所做的工作实际上就是记录对应位置上是否有某个字母，这让我想起来位操作上的某个位置为0或者1，而32位机器int的大小一般都是32bit，这么一来直接使用int存储不是很好吗？可以节约大量空间，判断是否存在相同字母的时候可以使用与操作，直接使用位操作的效率可是很高的，这么一来我就写出了使用位操作的代码。超过了80%的提交。

代码实现

最终的代码实现就是下面的了：

using ll = long long;
using ull = unsigned long long;
using uint = unsigned int;

class Solution {
public:
    int maxProduct(vector<string> &words) {
        // 1或0个返回0
        if (words.size() <= 1) {
            return 0;
        }
        int max = 0;
        ull size = words.size();
        // 使用uint的前26位记录字母是否存在
        vector<uint> check_chars(size, 0);
        for (int i = 0; i < size; ++i) {
            for (const char &ch : words[i]) {
                // 或运算
                check_chars[i] |= 1<<(ch - 'a');
            }
        }

        for (int i = 0; i < size - 1; ++i) {
            for (int j = i + 1; j < size; ++j) {
                // 检测是否有相同字母
                // 与运算
                if ((check_chars[i] & check_chars[j]) == 0 && words[i].length() * words[j].length() > max) {
                    max = words[i].length() * words[j].length();
                }
            }
        }

        return max;
    }
};
